# 🚌 Smart Transport Management System

A full-stack intelligent transport management platform designed to streamline premium intercity bus operations, passenger ticketing, fleet monitoring, and operational analytics.

---

## ✨ Features

### 🚌 Fleet Management
- Bus registration and management
- Route management
- Schedule creation
- Driver and conductor management
- Staff assignment management

### 🎟️ Passenger Operations
- Passenger registration
- Ticket booking system
- QR code ticket generation
- Duplicate seat booking prevention
- My Tickets page with search functionality

### 📍 Live Operations
- Live trip monitoring
- GeoFence route boundary checking
- Real-time transport tracking
- Incident reporting

### 🤖 Smart Intelligence
- Predictive maintenance insights
- Driver fatigue detection
- Route optimization recommendations
- Automated operational alerts

### 📊 Reporting & Analytics
- Revenue reports
- Fuel cost tracking
- Maintenance cost reporting
- Fleet performance dashboards
- Printable PDF operational reports

### 🔐 Role-Based Access
- Admin dashboard
- Driver dashboard
- Conductor dashboard
- Passenger portal

## 🚀 Installation

### 📥 Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/smart-transport-management-system.git
cd smart-transport-management-system
```

### ⚙️ Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

Run backend

```bash
npm run dev
```

---

### 🎨 Frontend Setup

```bash
cd frontend
npm install
```

Create `.env`

```env
VITE_API_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```

Run frontend

```bash
npm run dev
```
